
# E-Commerce Data Analysis (Olist)

## Project Overview
This project analyzes the Brazilian E-Commerce dataset provided by Olist. The goal is to understand customer behavior, sales performance, delivery efficiency, and business trends using data analysis.

The analysis focuses on generating meaningful insights that help improve business decisions.

---

## Objectives
- Analyze overall sales and revenue trends
- Identify top-performing product categories
- Understand customer purchasing behavior
- Study delivery performance and delays
- Perform customer value and repeat purchase analysis

---

## Dataset

This project uses the **Brazilian E-Commerce Public Dataset by Olist**.

Due to the large size of the dataset, the data files are not included in this repository.

You can download the full dataset from the link below:

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset includes multiple tables such as:
- Orders
- Customers
- Order Items
- Products
- Sellers
- Payments
- Reviews
- Geolocation

## Analysis Performed
- Monthly order trend analysis
- Revenue by product category
- Order value distribution
- Delivery time analysis
- Repeat customer analysis
- Customer lifetime value
- Top 10% customer revenue contribution
- Day-wise order distribution
- Correlation heatmap

---

## Key Insights
- A small percentage of customers contribute a large share of total revenue.
- Certain product categories drive the majority of sales.
- Most deliveries are completed within the estimated time, with some delays.
- Order values vary significantly across customers.
- Repeat customers play an important role in revenue growth.
